# Cloud Networking Drill 3: AWS VPC, Security Groups, NACLs, And Routing

## Goal

Be able to explain AWS networking without flattening it into generic cloud networking.

## Scenario

Design networking for a service in AWS using:

- Route 53
- ALB
- private app subnets
- private database subnets
- security groups
- NACLs

## Tasks

1. Explain where route tables apply.
2. Explain how SGs and NACLs differ operationally.
3. Explain what internet-facing versus internal ALB changes.
4. Explain one failure caused by SGs and one by NACLs.
5. Explain how NAT and egress affect private workloads.

## Model-Answer Rubric

- does the answer treat SG and NACL differences correctly
- does it connect routing to subnet design
- does it show real AWS reasoning instead of generic cloud abstractions

# Linux Admin Drills

These drills are for Linux command mastery and administration habits. They are intentionally practical and repetitive because senior confidence comes from fluency, not from reading command lists once.

## Drills

1. [drill-01-service-and-systemd-triage.md](drill-01-service-and-systemd-triage.md)
2. [drill-02-filesystem-and-storage-admin.md](drill-02-filesystem-and-storage-admin.md)
3. [drill-03-process-socket-and-network-admin.md](drill-03-process-socket-and-network-admin.md)
4. [drill-04-command-mastery-checklist.md](drill-04-command-mastery-checklist.md)

## Where To Run These

- local Linux host
- cloud VM
- Linux container with sufficient tooling for basic drills

## Mentor Tip

Do not try to memorize command syntax in isolation. Attach each command to a class of problem it helps solve.
